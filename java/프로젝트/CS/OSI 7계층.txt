7계층

사용자가 직접 볼 수 잇는 부분
최종 사용자에 가장 가까운 계층

정보 입력 -> 특정 프로토콜 형식으로 표현 -> 웹서버나 브라우저에 전송을 위한 계층

프로토콜 초기 : 'www' 상의 하이퍼 텍스트 -> 현재는 모든 형식의 데이터 전송 가능

OSI 7계층이 이론면에서는 좋지만, 실제로는 TCP/IP 계층이 많이 쓰인다.
